Portfólio - V1.2

Arquitetura Model View Template

Esta aplicação foi desenvolvida com Django e segue a arquitetura Model-View-Template (MVT). O objetivo foi aplicar os conhecimentos adquiridos nas aulas e também conhecimentos externos para construir um portfólio completo e funcional.

Modelação

O modelo de dados foi desenhado para representar as várias componentes do portfólio, incluindo licenciaturas, unidades curriculares, projetos, tecnologias, competências e outros elementos.

A modelação foi implementada no ficheiro models.py, utilizando classes Django. Cada classe corresponde a uma tabela na base de dados e define as relações entre os diferentes elementos.

Tecnologias da Aplicação

Foram utilizadas várias tecnologias no desenvolvimento desta aplicação.

  • Frontend: HTML, CSS
  • Backend: Python, Django
  • Base de dados: SQLite
  • Outros: Git, GitHub, Codespaces
